# Security Policy

Please do not report vulnerabilities through public GitHub issues.

Report security concerns to:

- [support@sciencebrew.com](mailto:support@sciencebrew.com)

Include:

- affected file or area
- reproduction steps if available
- severity/impact assessment
- whether the issue is public-repo-only or likely affects the hosted product

## Scope

This repository contains public integration assets only. Some security issues may
actually belong to the private RelayNet/OpenChat product rather than this public
repo. When in doubt, report it privately.
